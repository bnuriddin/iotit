Flashing Tool for IoT devices used by Isaax Cloud

license: Apache License Version 2.0

Overview

IoTit is a cross platform tool for flashing linux based SBC's safely and easily written in golang. It flashes via virtual machine to keep your machine safe and decluttered. It currently supports Raspberry Pi, Intel Edison, Nano Pi and BeagleBone with more planned to come. It is open source and licensed under the Apache 2.0 license.

IoTitフラッシュツールは VirtualBox を採用したコマンドラインユティリティです。IoTitを使うことによって、RaspberryPi、Intel Edison、Beaglebone、NanoPi のようなLinux系のシングルボードコンピュータを簡単に初期化することができます。
